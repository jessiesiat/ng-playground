Custom Filters

To create a new filter, will create ngFilters module and from there will register our custom filter.

js/filters.js var ngFilters = angular.module('ngFilters', []); ngFilters.filter('expression', function() { return function(input) { return input ? ':)' : ':('; } });

The name of our filter is expression and the function evaluates whether true or false, if true will return smiley face :) else frown :( face.

Template

Will also include our js/filters.js in our script tag

.. <script src="js/controllers.js"></script> <script src="js/filters.js"></script> ... <body ng-controller="UserCtrl"> <ul> <li ng-repeat="user in users"> <strong>{{ user.happy | expression }} {{user.name}}</strong> - <em>{{user.email}}</em> </li> </ul> </body>

Now that our filter is ready will now include inject the ngFilters module as dependency to our main ngPlayground module.

var ngPlayground = angular.module('ngPlayground', ['ngFilters']);

There are built in filters of Angular just like currency, uppercase, date, orderBy etc. Feel free to explore!
