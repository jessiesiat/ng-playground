Bootstraping

First of all will include the angularjs library at the bottom of the html page.

<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/angularjs/1.2.1/angular.min.js"></script>

To bootstrap the page will placed the ng-app attribute which represents Angular ngApp directive. Will name it ngPlayground so that we can reference it later on in our code.

<html ng-app="ngPlayground">

We have placed it in our html element which means that the whole html page will be considered as Angular application. You can place it to any element if you want only portion of the page to be an Angular application.

Our starter template would look something like this!

<!doctype html> <html lang="en" ng-app="ngPlayground"> <head> <meta charset="utf-8"> <title>Ng Playground</title> <script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/angularjs/1.2.1/angular.min.js"></script> </head> <body> <p>{{'Templates here' + '!'}}</p> </body> </html>
